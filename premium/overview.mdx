Sub offers two tiers: a free tier that covers everything you need to track subscriptions locally, and a Premium tier designed for power users and developers who need greater API capacity. Both tiers include the full set of core features — Premium simply raises the ceiling on how you can interact with your data programmatically.

## Feature Comparison

| Feature | Free | Premium |
|---|---|---|
| Active API keys | 1 | 5 |
| API requests per hour | 60 | 1,000 |
| Cloud sync | ✓ | ✓ |
| Bark renewal reminders | ✓ | ✓ |
| Import / export | ✓ | ✓ |
| Analytics | ✓ | ✓ |

<Note>
  Every core feature — subscription tracking, analytics, categories, import/export, and Bark push notifications — is fully available on the free tier. Upgrading to Premium only expands your API key and request-rate allowances.
</Note>

## How Premium Works

Premium is a **one-time purchase** processed securely through Stripe Checkout. There are no recurring charges or subscriptions — you pay once and your account is upgraded immediately after Stripe confirms the payment.
